# 🚀 Quick Start Guide

## Welcome to CXRT AI Web Application!

This is your complete guide to getting started with the web version of the CXRT AI application.

---

## ⚡ 3-Step Quick Start

### 1️⃣ Open the Application
The app starts automatically on **Screen 1** (Splash Screen)

### 2️⃣ Navigate to Login
- **Splash Screen** → Click anywhere
- **Welcome Screen** → Click "Get Started"

### 3️⃣ Choose Your Role
Select one of three roles:
- **👨‍⚕️ Doctor** → Medical diagnosis and case management
- **👨‍🔧 Technician** → X-ray scanning and patient registration
- **👔 Admin** → System administration

> **Note**: For demo purposes, any username/password combination will work!

## 🆘 Troubleshooting

### Can't Login?
- **Any username/password works!** (Demo mode)
- Make sure you selected a role
- Try refreshing the page

### Screen Looks Broken?
- Zoom level: Reset to 100% (Ctrl+0)
- Browser: Try Chrome/Edge
- Clear cache: Ctrl+Shift+Delete

### Navigation Not Working?
- Click the hamburger menu (☰) on mobile
- Use sidebar on desktop
- Refresh if stuck

### Images Not Loading?
- Check internet connection
- Allow images in browser settings
- Try refreshing the page
